# FAQ

**Q: Why did you make this app?**

**A:** Not a big fan of IDEs (ex: GOWIN) other than VS Code. So I had to look for open source options to run code on my dev board. Long story short, I did not look for alternatives and started to make a small app which automates a few commands to build the verilog code I wrote with some "this would be a nice feature" features in it.

**Q: What boards does the app support?**

**A:** [Tang Nano 9K](https://wiki.sipeed.com/hardware/en/tang/Tang-Nano-9K/Nano-9K.html).

**Q: Why just one board?**

**A:** Tang Nano 9K is the first and only FPGA dev board I have. I cannot assure if the app will work for other boards. [Feel free to contribute if you have other boards](CONTRIBUTING.md).

**Q: On what OS I can run this app?**

**A:** Windows

**Q: Why just Windows?**

**A:** I *only* use a Windows machine. I cannot assure if the app will work on other OS. [Feel free to contribute if you manage to make the app work on Mac or Linux](CONTRIBUTING.md).

**Q: 🙋 I have more questions**

**A:** Ask them at the [discussions](https://github.com/biologyscience/fpga-toolkit/discussions).
