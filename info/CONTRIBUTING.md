# How do I contribute?

You can contribute in three different ways:
- Report a bug or unintended behavior.
- Suggest a feature.
- Share working piece of code which adds/improves a feature.

## Report a bug or unintended behavior
Create a Issue [here](https://github.com/biologyscience/fpga-toolkit/issues).

## Suggest a feature
Add them in [discussions](https://github.com/biologyscience/fpga-toolkit/discussions).

## Code contributions
Start a [pull request](https://github.com/biologyscience/fpga-toolkit/pulls).

Make sure you mention the following in your pull request:
- Why make this pull request?
- Does it affect a existing feature?
- Proof (screenshot/video/etc.) of functional use of your pull request.
