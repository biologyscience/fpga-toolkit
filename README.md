# FPGA Toolkit
A small wrapper to automate the build process of generating a bitstream from a verilog file.

# About
- Backend with Electron.
- UI with React.
- Supports **ONLY** [Tang Nano 9K](https://wiki.sipeed.com/hardware/en/tang/Tang-Nano-9K/Nano-9K.html) !
- Spent just **one** day to create this app. Beware of bugs and unintended behavior.
